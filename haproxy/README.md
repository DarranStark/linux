## HAProxy 是什么？
  **HAProxy**是一款自由，快速，可靠的解决方案。它用以提供高可用性、负载均衡和基于tcp和http应用的代理。  

## HAProxy 有什么特点？
  * 支持虚拟主机，工作在4，7层
  * 支持url检测后端服务器
  * 更强大的session保持，cookie的引导
  * 支持多的负载均衡算法
