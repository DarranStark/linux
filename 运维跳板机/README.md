## 运维跳板机  
* AAAA认证即：  
  统一用户账号（Account）
  统一认证（Authentication）
  统一授权（Authorization）
  统一安全审计（Audit）  

* 使用开源软件实现AAAA认证系统  
openldap(管理用户信息，实现资源访问控制)+Kerberos(网络鉴权)+google authenticator(二次认证)+bash(修改源码，实现远程用户的操作日志记录)
